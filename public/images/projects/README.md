# Project images

Add at least **3 images per project** in:

- `public/images/projects/<slug>/`

Recommended naming:

- `01.jpg`
- `02.jpg`
- `03.jpg`

You can also add more (e.g. `04.jpg`, `05.jpg`).

Examples (current slugs):

- `public/images/projects/snoefjell/01.jpg`
- `public/images/projects/malm-fosdalen/01.jpg`
- `public/images/projects/lergruvbakken/01.jpg`
