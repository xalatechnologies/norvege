# Investors Page Images

This folder contains images for the Investors page.

## Naming Convention

- `hero.jpg` - Main hero image for the Investors page hero section

## Image Requirements

- **Format**: JPG or PNG
- **Recommended size**: 1920x1080px or larger (16:9 aspect ratio recommended)
- **Quality**: High resolution for best display on all screen sizes
- **File size**: Optimize images to keep file sizes reasonable (ideally under 500KB)

## Usage

The hero image will be displayed as a full-background image in the Investors page hero section with text overlaid on top.
