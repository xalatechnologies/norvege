# Hero Section Images

This folder contains images for the hero section carousel on the homepage.

## Naming Convention

Images should be named sequentially:

- `01.jpg` - First hero slide (Responsible Mining for the Green Transition)
- `02.jpg` - Second hero slide (Gaulstad/Mokk)
- `03.jpg` - Third hero slide (Rare Earth Elements)
- `04.jpg` - Fourth hero slide (Tr√∏ndelag Region)
- `05.jpg` - Fifth hero slide (Science-Driven Discovery)

## Image Requirements

- **Format**: JPG or PNG
- **Recommended size**: 1920x1080px or larger (16:9 aspect ratio)
- **Quality**: High resolution for best display on all screen sizes
- **File size**: Optimize images to keep file sizes reasonable (ideally under 500KB per image)

## Usage

The hero section will automatically use these images in order. If an image is missing, the component will fall back to the gradient background defined for that slide.
