# Logos Directory

This directory contains logo files for various partners and integrations.

## NorChain Logo

- `norchain.png` or `norchain.svg` - Logo for NorChain blockchain platform

## Logo Guidelines

- Use PNG format for logos with transparency
- Use SVG format for scalable vector logos
- Recommended size: 64x64px minimum, 128x128px preferred
- Ensure logos are optimized for web use
