# Videos Directory

This directory contains video assets for the website.

## Hero Video

- `mining.mp4` - Main hero background video showing minerals inside mountains (used as first slide in hero carousel)

## Video Guidelines

- Use MP4 format for maximum browser compatibility
- Keep file sizes optimized for web (consider compression)
- Videos should be muted and looped for background use
- Recommended resolution: 1920x1080 or higher
- Recommended codec: H.264
